\# Technology Stack

\## Frontend Architecture

- React (with Vite or Next.js)

- Tailwind CSS (fast UI)

- Chart.js or Recharts (charts)

\## Login / Registration

- Email / Phone login

- JWT based -auth

\## Configuration Page

- Nifty 100

- Duration \< 120 days

- Trigger Frequency

\## Screens

- Login Page

- Home Dashboard

- Configuration Page

- Stock Page

\## Backend Architecture

\### Tech Stack

- Python

- FastAPI

- PostgresSQL

- SQLAlchemy(ORM)

- Auth Service

  - Login / Register

  - JWT Tokens

- Config Service

  - Save user config

  - Yahoo Finance API calls

  - Incremental fetch

- Analysis Engine

  - Compute gainers / losers

  - Trend classification

  - Weekly Aggregation

- News Service

  - Scrape news

  - Store headlines + links

- Job scheduler

  - Run weekly jobs
